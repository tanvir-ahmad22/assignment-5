Question-1: What are the different ways to select an element in the DOM?

Elements can be selected using getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets content with HTML tags, innerText gives only the visible text, and textContent gets just the text without HTML.

Question-3: What is event delegation in the DOM?

Event delegation is when an event listener is added to the parent element to handle events for child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when the event moves from the child element to the parent.

Question-5: How do you create, add, and remove elements using JavaScript?

createElement() is used to create an element, appendChild() is used to add it, and removeChild() is used to remove it.
